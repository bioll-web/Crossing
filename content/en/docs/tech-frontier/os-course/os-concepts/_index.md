# 《操作系统概念》深入阅读

教材：Silberschatz, Galvin, Gagne —《Operating System Concepts》第 9 版（恐龙书）。

这条线的目的不是"把教材抄一遍"，而是**用教材的概念骨架去反向校准源码阅读**。读源码容易陷进"这一行代码为什么这么写"的细节里，教材提供的是"为什么整个子系统要这么设计"的视角，两者必须交叉。

每章笔记的写法约定：

1. 先用自己的话复述核心概念（不回看书）
2. 再对照教材检查有没有偏差
3. 最后补一条"这个概念在 Linux 0.01 里对应哪段代码"
