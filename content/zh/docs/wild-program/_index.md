# Wild Program · 野生程序

> 一些不为作业、不为绩点、只为好玩的程序

这里收录我在课业之外写的小程序、小工具、小实验 ——
它们大多没有实际用处，却是我学习过程中最诚实的副产物。

「野生」两个字，是想说：
这些代码不生于课程大纲，也不为期末考试服务，
它们只是在某个夜里被好奇心逼出来的东西。

## 可能会出现的内容

- 命令行小工具
- 算法可视化实验
- 解题笔记与代码
- 临时起意的练手项目

每个项目会附带 GitHub 源码链接，欢迎 fork、改、骂。

---

*野生的东西长势不定，更新也不定。*
