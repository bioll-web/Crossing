# Classic Talk · 经典漫谈

> 在技术之外的另一半我

这里记录我读过的书、看过的电影，
以及它们在某个深夜里让我忽然安静下来的瞬间。

工科人读文学并非跨界猎奇 ——
一个系统如何运作，和一个人如何存在，
本质上是同一类追问。

## 关注的方向

- **古典文学** · 《诗经》、古诗十九首、陶渊明、苏轼
- **哲学随笔** · 加缪、韦伯、林语堂
- **现代文学** · 余华、卡夫卡
