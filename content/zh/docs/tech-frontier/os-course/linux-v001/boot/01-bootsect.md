# bootsect.s：第一个 512 字节

BIOS 在完成自检后，会把磁盘第 0 扇区（正好 512 字节）加载到物理地址 `0x7C00` 并跳过去执行。Linux 的这个扇区做了三件事：

1. 把自己搬到另一个地址（避开后续要加载的 setup）
2. 调 BIOS 中断把 setup 和内核本体读进内存
3. 跳转到 setup 的入口
