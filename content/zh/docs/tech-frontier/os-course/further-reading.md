# 延伸阅读与资源索引

## 在线源码浏览

- [elixir.bootlin.com](https://elixir.bootlin.com/) — 各版本 Linux 源码，可跨文件跳转
- [TUHS](https://www.tuhs.org/) — Unix 历史源码归档（含 V6、V7）
- [xv6-riscv on GitHub](https://github.com/mit-pdos/xv6-riscv)

## 经典书籍

- *Operating System Concepts*（Silberschatz）— 概念骨架
- *Operating Systems: Three Easy Pieces*（Remzi）— 通俗易懂，免费在线版
- *Modern Operating Systems*（Tanenbaum）— 偏系统视角
- 《Linux 内核设计的艺术》— 对应 Linux 0.11 的中文讲解
- 《深入理解 Linux 内核》— 现代 Linux 的百科全书

## 课程

- MIT 6.828 / 6.S081 — <https://pdos.csail.mit.edu/6.828/>
- 清华 uCore — <https://learningos.github.io/uCore-Tutorial-Guide-2022A/>
- rCore（Rust 版 OS 教程）— <https://rcore-os.github.io/rCore-Tutorial-Book-v3/>

## 实用工具

- QEMU — 全系统模拟器
- GDB + `-s -S` QEMU 参数 — 远程调试内核
- `bochs` — 可视化调试，适合看实模式
